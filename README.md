# mipt-secure-sql-agents

Мультиагентная система, которая генерирует запросы PostgreSQL на естественном языке, проверяет их на наличие проблем с безопасностью и итеративно устраняет их с полным протоколированием аудита. Разработана в рамках проекта практического курса МФТИ.

## Структура проекта

```sh
.
├── app         # код приложения
├── docs        # некоторые документы
└── notebook    # исследовательские jupyter notebook
```

## Начало работы

1. Создать venv и активировать:

    ```sh
    python3 -m venv .venv
    source .venv/bin/activate
    ```
2. Установить зависимости из requirements.txt:

    ```sh
    pip install -r requirements.txt
    ```

3. Создать локальный .env файл с секретами для запуска приложения:

    ```sh
    OPEN_ROUTER_API_KEY=<your_api_key>
    MODEL_NAME=openai/gpt-4o-mini
    PORT=8000
    ```

## Запуск

```sh
docker compose up
```

Код монтируется из текущей директории — сервер перезапускается при изменении файлов. Переменные окружения читаются из `.env`.
